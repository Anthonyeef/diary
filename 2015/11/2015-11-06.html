Jekyll

I read the book about jekyll today, and start a repo locally. May push it to GitHub later.

Here is my notes.

  • Every post in the folder _post contains the name made up with date,slug and extension. For example, 2015-11-06-welcom-to-jekyll.markdown.

  • Rake in Ruby is the tool like make in C++. The action of "rake post" is defined in Rakefile.

  • The default way of the post link is date, which looks like: /:categories/:year/:month/:day/:title/. And there is another two kinds of link style. pretty looks like: /:categories/:year/:month/:day/:title/, the none looks like: /:categories/:title.html.
